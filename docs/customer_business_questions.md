# Hi-D Search(Data) 연동을 위한 확인사항

## 목적

Hi-D Search(Data) API (Application Programming Interface: 프로그램 간 통신 인터페이스)를 Spotfire Copilot에 성공적으로 연동하기 위해 사전에 확인이 필요한 항목들을 정리한 문서

**Hi-D Search(Data)란?**
- LG디스플레이의 지표 데이터 조회 시스템
- 생산·품질·공정 등 분석 데이터 제공
- 실시간 데이터 접근 및 활용 가능
